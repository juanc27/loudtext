Hi, Welcome to loud text app


This app measures the input volume of your mic and send text messages to a phone in the US when :

* The input volume crosses a minimum threshold for more than 1 second.

AND

* While the input volume is above the minimum threshold for more than one second, the cumulative volume output (volumeÂ­seconds) is one of the top 3 highest output time periods recorded since the start of observation. Where output is defined as output= sum(volume*delta(time))

The text message will be send after the volume lowers (crosses) down the minimum threhold
